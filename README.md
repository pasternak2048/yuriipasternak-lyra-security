## ğŸ” Signing Flow (Client-side)

1. Hash the payload body (e.g. JSON) using SHA-512
2. Build a `StringToSign` (method + path + hash)
3. Compute HMAC-SHA512 using a secret key
4. Create a `VerifyRequest` and send to verifier
